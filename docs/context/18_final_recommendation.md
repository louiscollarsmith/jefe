# 18 — Final Strategic Recommendation

Build a bounded, evidence-first operator for Shopify.

Ship it in a week.
Harden it in production.
Verify it at BFCM.

Architecture:
- deterministic workflow engine
- approval-gated execution
- House Rules
- caps
- event ledger
- provenance
- holdouts

Self-learning:
- action ranking
- attribution
- calibration
- feedback-to-roadmap

Not:
- unconstrained autonomy
- generic chatbot
- unbounded self-modifying agent

Judge the product on one question:

> Did it deliver holdout-verified incremental margin for the merchant this month?

Shopify is the wedge.

Autonomous, accountable commerce operations is the long-term platform story.
